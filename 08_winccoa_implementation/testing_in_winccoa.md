# Event Handling

## Principle (EN)
System reacts to changes in datapoints.

## Принцип (RU)
Система реагирует на изменения datapoint.

---

## Flow
1. switching device state changes
2. topology recalculation
3. update visualization

## Поток
1. изменение состояния аппарата
2. пересчёт топологии
3. обновление отображения
