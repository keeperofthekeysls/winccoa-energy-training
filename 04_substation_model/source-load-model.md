# Source and Load

## Source (EN)
Element that provides power.

Examples:
- External grid
- Generator
- Battery system (DC supply)

## Источник (RU)
Элемент, подающий питание.

Примеры:
- Внешняя сеть
- Генератор
- Аккумуляторная батарея (система постоянного тока)

---

## Load (EN)
Element that consumes power.

## Нагрузка (RU)
Элемент, потребляющий электроэнергию
