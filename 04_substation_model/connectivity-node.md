# Connectivity Node

## Definition (EN)
ConnectivityNode represents a connection point between terminals.

## Определение (RU)
ConnectivityNode — это точка соединения терминалов. ConnectivityNode = вершина графа / узел / участок цепи.
ШР-1 = оборудование / ребро
Проект/VL3/Q1/L2 = вершина / ConnectivityNode
Проект/VL3/W0/B0 = вершина / ConnectivityNode

Если <Bay> содержит только <ConnectivityNode>, а к этому <ConnectivityNode> подключено много <Terminal> от разного оборудования, то в графе это: одна вершина с большим числом подключенных ребер
---

## Role
- Graph node
- Electrical junction

## Роль
- Вершина графа
- Электрический узел
