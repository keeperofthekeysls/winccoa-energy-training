# Topology Coloring Task

## Goal (EN)
Implement dynamic detection of energized sections in a substation and visualize them in SCADA.

## Цель (RU)
Реализовать динамическое определение участков под напряжением на подстанции и их визуализацию в SCADA.

---

## Input Data
- Substation topology (nodes and connections)
- Switching device states
- Power sources

## Входные данные
- Топология подстанции (узлы и связи)
- Состояние коммутационных аппаратов
- Источники питания

---

## Output
- Each section is marked as:
  - energized
  - de-energized

## Выход
- Каждый участок имеет состояние:
  - под напряжением
  - без напряжения
