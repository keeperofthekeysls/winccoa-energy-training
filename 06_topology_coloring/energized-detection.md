# Energized Detection

## Principle (EN)
A node is energized if there exists a path from a power source through closed connections.

## Принцип (RU)
Узел считается под напряжением, если существует путь от источника через замкнутые соединения.

---

## Approach
- start from sources
- traverse graph
- mark visited nodes

## Подход
- старт от источников
- обход графа
- маркировка узлов
