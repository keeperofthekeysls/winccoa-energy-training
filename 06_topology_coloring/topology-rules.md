# Topology Model

## Graph Representation (EN)
- Nodes: connectivity nodes
- Edges: electrical connections (through equipment)

## Представление графа (RU)
- Вершины: ConnectivityNode
- Рёбра: электрические соединения (через оборудование)

---

## Important Rules
- Connection exists only if switching device is closed
- Graph is dynamic

## Важные правила
- Связь существует только при замкнутом аппарате
- Граф динамический
